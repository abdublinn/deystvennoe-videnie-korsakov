# Лендинг «Действенное видение»

> Редакция Сергея Корсакова на базе общего лендинга.

Статический адаптивный лендинг без сборки и внешних зависимостей.

## Предпросмотр

Можно открыть `index.html` двойным щелчком. Для полной проверки формы лучше запустить локальный сервер:

```powershell
python -m http.server 4173
```

И открыть `http://localhost:4173`.

## Перед публикацией

1. Уточнить дату ближайшей сессии. Сейчас на странице указано: «по мере набора группы», так как дата 19 марта из исходника уже прошла.
2. Если меняется маршрут заявок, обновить в `script.js` адрес `LANDING_CONFIG.submitEndpoint`. Сейчас форма подключена к `FormSubmit` через AJAX relay на `skorsakov.spb@gmail.com`.
3. После первого боевого подключения подтвердить письмо активации сервиса у Сергея, иначе заявки будут ждать верификации адреса.
4. Проверить цену и формулировки об экспертах.

Без `submitEndpoint` форма работает в fallback-режиме: проверяет поля и открывает предзаполненное письмо на адрес получателя.

## Файлы

- `index.html` — структура и контент.
- `styles.css` — визуальная система и адаптив.
- `script.js` — анимации, FAQ, валидация и отправка формы.
- `assets/experts` — облегчённые веб-версии портретов; исходные JPG остаются нетронутыми.
